# 2048_M5Stack
## M5Core2ライブラリプロジェクト
M5Core2のSD-Updaterを用いたライブラリ集を構築するプロジェクトの一環として実装されています。

## どんなアプリ？

M5Stack-Core2用のArduinoコードで、RTCの時刻をデジタルまたはアナログ形式で表示します。

このアプリは、もともとM5Core2用となっており、新たにSD-Updaterに対応しました。

ボタンを押すと、デジタルとアナログ時計を切り替えます。内蔵のRTCクロックを用います。
